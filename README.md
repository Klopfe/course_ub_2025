# Course UB 2025

## Description

This is the repository for a 3 hours workshop on useful tools for data science given to Master MIGS 2nd year.

## Content

- Notebook to implement XGBoost for regression
- Notebook to have a look at the data
- Notebook to compare the performance of the different models

## Clone the repository

```bash
git clone https://github.com/Klopfe/course_ub_2025.git
cd course_ub_2025
```

## Python package manager: `uv`
[`uv`](https://docs.astral.sh/uv/) is a package manager for Python. It is a modern, fast, and secure package manager for Python. It is a drop-in replacement for `pip` and `conda`.

## Installation

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## Install the dependencies

```bash
uv sync --all-extras
```
